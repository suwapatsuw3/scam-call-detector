ARCHITECTURE DECISION

This is a Third party application using CALLER speech to Identify as if he/she is scammer or not. using LangGraph Agentic AI and AI Agent with SCAM message database to detect scammer

LOCAL

"All models run locally; there is no need to send data to the cloud, so there is no risk of data theft."

After receiving CALLER voice from phone speaker, the system segments the voice stream using Silero VAD and converts it to text via quantized distill-whisper-th ASR. The content then undergoes classification by WangchanBERTa, if a 'SCAM' intent is identified 3 times, Qwen3-1.7B will formulate persuasive Warning and alerts the user.

Model/Tools used

  • Silero VAD (2MB) : utilizes a deep neural network to accurately distinguish human speech from background noise
  • distill-whisper-th-small (0.2B, Quantized) : whisper-medium with Thai finetuned
  • wangchanberta-base-att-spm-uncased (0.1B, Quantized, finetuned) : suitable for Thai text classification
  • Qwen3-1.7B : small and intelligent enough to give sensible reason

* It needs BERT to trigger SCAM 3 times (1-2 full sentences) to activate Qwen to prevent phone from overheating

AI Agent on Cloud

"Need Internet to activate this system. Longer thought, safer recomendation"

This Cloud-based AI Agent operates via a Decision Orchestrator powered by NVIDIA-Nemotron-3-Nano-30B-A3B-FP8. It ingests the conversation text from ASR and initial 'SAFE' classification from wangchanBERTa to re-evaluate the verdict. If model is UNSURE, it will gathers more information by triggering RAG (to retrieve previous scam patterns and chat history) and Google Search API for external data. Finally, if the Orchestrator identifies the call as a scam with High Confidence, it immediately alerts the user.

Model/Tools used

  • NVIDIA-Nemotron-3-Nano-30B-A3B-FP8 : MOE model; suitable with long context, Low latency High throughput

Hybrid RAG

Use for

  • Managing memory, Collecting chat history
  • Retrieving the messages that labeled as SCAM
  • Retrieving past SCAM detected message globally
  • Finding semantically similar scam scripts
  • Refining context relevance with Reranker
